## 1. Context

Consider the irreversible reaction from reactant $A$ to product $B$

$$
A \longrightarrow B
\tag{1}
$$

with the first-order reaction rate law

$$
r_A = -k(T) C_A
\quad ,
\tag{2}
$$

where $r_A$ is the rate of consumption of species $A$, $k(T)$ is the rate constant, and $C_A$ is the concentration of species $A$.  The temperature dependence of the rate constant is given by the Arrhenius law

$$
k(T) = k_0\,\exp\!\left(-\frac{E_a}{RT}\right)
\quad ,
\tag{3}
$$

where $k_0$ is a constant, $E_a$ is the activation energy associated with the reaction, and $R$ is the ideal gas constant.

In a continuously-stirred tank reactor (CSTR), the dimensionless conversion $X$ of the reactant $A$ is commonly expressed as

$$
C_A = C_{A0} (1 - X)
\quad\text{with}\quad
X = \frac{k(T)\, \tau}{1 + k(T)\, \tau}
\quad ,
\tag{4}
$$

where $C_{A0}$ is the inlet concentration of the reactant species $A$ and $\tau$ is the *space time*, i.e., the ratio $\tau = V/v_0$ of the reactor volume $V$ and the volumetric flow rate $v_0$.

In design, control, and safety discussions, an important local sensitivity is

$$
\frac{dX(T)}{dT}\bigg|_{T=T_0},
\tag{5}
$$

which quantifies how strongly conversion responds to small temperature perturbations.

In realistic ChemE workflows, $X(T)$ is often obtained from numerical models (e.g., coupled mole+energy balances, multiple reactions, non-integer orders, non-ideal kinetics). In those cases, analytic derivatives are unavailable or impractical, and sensitivities are estimated numerically.

The *central difference* approximation is a common approach for numerical differentiation

$$
\frac{df(x)}{dx}\bigg|_{x=x0}
\approx
D_c(h) = \frac{f(x+h) - f(x-h)}{2h}
\quad ,
\tag{6}
$$

which has an expected truncation error of order $O(h^2)$ in the step size $h$.

## 4. Implementation results

The following code was generated by ChatGPT provided a prompt composed exclusively of the above *objective*, *constraints*, and *approach*. 

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# -----------------------------
# Given parameters
# -----------------------------
k0 = 1.0e6       # 1/s
Ea = 80000.0     # J/mol
R  = 8.314       # J/(mol*K)
tau = 2.0        # s

# -----------------------------------------
# Step 1: Implement X(T)
# X(T) = k(T)τ / (1 + k(T)τ),  k(T)=k0 exp(-Ea/(RT))
# -----------------------------------------
def k_arrhenius(T, k0=k0, Ea=Ea, R=R):
    """Arrhenius rate constant k(T) [1/s]."""
    T = np.asarray(T, dtype=float)
    return k0 * np.exp(-Ea / (R * T))

def X_CSTR(T, tau=tau, k0=k0, Ea=Ea, R=R):
    """CSTR conversion X(T) for first-order kinetics."""
    k = k_arrhenius(T, k0=k0, Ea=Ea, R=R)
    kt = k * tau
    return kt / (1.0 + kt)

# --------------------------------------------------------------------
# Step 2: Central difference derivative
# D(h) = [f(T+h) - f(T-h)] / (2h)
# --------------------------------------------------------------------
def central_difference(f, T_values, h):
    """
    Central-difference approximation of df/dT on a grid.
    Parameters
    ----------
    f : callable
        Function f(T).
    T_values : array-like
        Temperatures [K] at which to evaluate derivative.
    h : float
        Step size [K] (must be > 0).
    """
    T_values = np.asarray(T_values, dtype=float)
    if h <= 0:
        raise ValueError("Step size h must be positive.")
    return (f(T_values + h) - f(T_values - h)) / (2.0 * h)

# ------------------------------------------------------
# Step 3: Evaluate derivative, table, and plot
# ------------------------------------------------------
Tmin, Tmax = 300.0, 500.0
n_points = 201
T_grid = np.linspace(Tmin, Tmax, n_points)

h = 0.5  # step size in K (adjust to test numerical sensitivity)

X_vals = X_CSTR(T_grid)
dXdT_vals = central_difference(X_CSTR, T_grid, h)

# Results table
df = pd.DataFrame({
    "T [K]": T_grid,
    "X(T) [-]": X_vals,
    "dX/dT [1/K] (central diff)": dXdT_vals
})

print(df.head(10))
print("\nSummary:")
print(df.describe())

# Plot dX/dT vs T
plt.figure(figsize=(7, 4.5))
plt.plot(T_grid, dXdT_vals)
plt.xlabel("Temperature, T [K]")
plt.ylabel("Sensitivity, dX/dT [1/K]")
plt.title("CSTR Conversion Sensitivity to Temperature (Central Difference)")
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()
```
